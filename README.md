### URL Analyzer & Content Classifier
---
## A Python-based tool that analyzes website URLs using Regular Expressions (Regex). The tool identifies duplicate URLs, categorizes URLs based on patterns, and generates reports for content quality analysis.
---
Tech Stack:
- Python
- Regex
- File Handling
- Data Analysis
---
# How To Execute

1) Change file permissions
```bash
chmod 775 analyzer.py
```

2) Execute
```bash
python analyzer.py
```
